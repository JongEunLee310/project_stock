# Claude Command: Create Codex Task

Create a Codex handoff task.

## Required Reading

- The source issue.
- `AGENTS.md`.
- `CLAUDE.md`.
- `docs/harness/handoff-policy.md`.
- `docs/harness/agent-role-policy.md`.
- `docs/knowledge/workflow.md`.
- Relevant knowledge base documents.
- `.codex/task-template.md`.

## Rules

- Do not edit source code.
- Produce the handoff using `.codex/task-template.md`.
- Include implementation scope.
- Include out-of-scope items.
- Include protected files.
- Include verification commands.
- Include documentation impact.
- Include ADR and failure record assessment.
- Include stop conditions when context is missing or risk is high.
