## 계약 일치 (docs/designs/price-series-api.md § 계약 확정)
- 경로 `/api/v1/stocks/{symbol}/prices`(prefix `/stocks`) — 일치.
- 쿼리 `market`(필수 Literal KRX|NASDAQ|NYSE)·`range`(alias, 기본 3M)·`interval`(1d)·`adjusted` — 일치. 잘못된 market은 FastAPI 422, range/interval은 명세 400.
- 응답 snake_case·OHLC+adjusted_close Decimal 문자열·`volume` 정수·`last_updated_at` UtcDatetime — 일치.
- **오프바이원 차단 검증 통과**: mock이 `datetime.combine(date, time.min, tzinfo=utc)`로 UTC 자정 생성, 응답 `date=timestamp.date().isoformat()`. tz-aware UTC라 서버 TZ 무관하게 거래일 날짜 보존. `TZ=UTC` 단언 통과.
- 에러코드 4종·HTTP 매핑 일치. 공개 GET(인증 없음 — OQ-3 확정대로, assets 시장데이터와 일관).

## 정확성 포인트
- `list_recent`: `timestamp.desc()` + limit 후 `reversed()` → 오름차순 반환. 테이블에 count 초과 적재돼도 최신 count개를 오름차순으로. 정확.
- `upsert_many`: 유니크키 조회 후 insert/update. 멱등성 테스트 통과(2회 호출 시 행 1개).
