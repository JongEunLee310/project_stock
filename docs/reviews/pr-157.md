# PR #157 — 시장 지수 시세 endpoint (feat/be-market-indices)

리뷰어: Claude Code (Opus, 오케스트레이터). 구현: Codex. 브랜치: feat/be-market-indices → main. 관련: BE #154, FE #87, 설계 061.

## Review Summary

설계 061대로 `GET /market/indices` endpoint를 추가했습니다. 인증 없이 대표 지수 4종(SPX·IXIC·KOSPI·VIX)의 현재값·전일 대비 변동률을 공통 envelope(`ApiResponse[list[MarketIndexQuoteResponse]]`)로 반환합니다. adapter 계층은 기존 seam 관례를 그대로 따라 frozen `@dataclass`(`IndexQuoteResult`)를 반환하고, `MarketIndexService`가 이를 응답 스키마로 매핑합니다. `MockIndexQuoteProvider`는 `sha256` 기반 결정론적 값과 고정 `_AS_OF` 기준 시각을 사용해 재현성을 확보합니다. factory `get_index_quote_provider()`는 기존 `MARKET_PROVIDER` 스위치를 재사용하며, 서비스는 provider 호출을 `try/except`로 감싸 실패 시 502(`MARKET_DATA_PROVIDER_ERROR`)를 던집니다. DB·마이그레이션은 없습니다.

설계의 확정 결정(데이터 소스=mock 우선 + seam, 지수 코드 집합=adapter 상수, 인증 불요, 스냅샷 조회)이 모두 반영됐고, 종목 quote(`MarketDataProvider`)와의 이름 충돌을 `IndexQuoteProvider`로 분리해 회피했습니다.

## Blocking

None.

## Suggestions

**S1 · 지수 코드 상수의 계층 위치 (비차단)**

`MARKET_INDEX_SYMBOLS` 상수가 `app/adapters/market/mock.py`에 있고, 도메인 서비스 `index_service.py`가 이를 mock 모듈에서 import합니다. 설계 §5가 "mock.py 또는 별도 상수 모듈" 배치를 허용했으므로 스코프 내이지만, 도메인 계층이 mock 구현에 결합되는 형태입니다. 외부 provider 연동 시점에는 코드 집합을 mock과 무관한 위치(예: `base.py` 상수 또는 config)로 옮기는 편이 결합을 줄입니다. 현 단계에서는 비차단입니다.
