## Suggestions

### 1. `check_concentration` 신호 기준이 원가→시세 비중으로 암묵 변경됨 (Minor)

**위치:** app/domains/portfolios/service.py:131-157

`PositionWeight.weight`가 이제 시세 기반이므로, 기존 `check_concentration`의 위험 신호 생성·`_score_from_weight` 입력 기준도 원가 비중에서 시세 비중으로 바뀐다. 의도된 개선으로 보이며 evidence에 `cost_value`/`market_value`를 모두 남겨 추적 가능하다. 다만 이슈 #54의 명시 범위는 아니므로, 기존 집중도 알림 동작에 의존하는 소비자가 있다면 동작 변화를 PR 설명에 한 줄 남겨두면 좋다.

**권장 수정 방향:** 동작 변경을 의도로 확정하고 PR/설계문서 Decisions에 "집중도 신호 기준을 시세 비중으로 통일" 명시. 코드 수정은 불필요.

### 2. 종목별 시세 조회 시 자산 단건 반복 조회 (Minor)

**위치:** app/domains/portfolios/service.py:198-205 (`_get_assets_by_position`)

`asset_repo.get_by_id`를 포지션 자산 수만큼 반복 호출한다. 현재 포트폴리오 규모에서는 문제 없으나, 향후 대량 포지션 시 N+1이 될 수 있다. 시세 조회(`get_quote`)는 이미 1회로 묶여 있어 양호하다.

**권장 수정 방향:** 현 단계에서는 불필요(과최적화 지양). 포지션 수가 커지면 `id IN (...)` 일괄 조회로 전환 검토.
