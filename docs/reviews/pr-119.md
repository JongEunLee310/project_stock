## 정확성

- **하위호환**: `expand` 미지정·지원 외 값이면 기존 `SignalResponse` 경로(추가 키 없음). 테스트 `assert "asset" not in data[0]`로 확인. ✔
- **expand 파싱**: watchlist와 동일 `"asset" in [e.strip() for e in expand.split(",")]` — `"metadata, asset"`도 정상 인식(테스트 커버). ✔
- **상속 보존**: `SignalExpandedResponse(SignalResponse)`가 `evidence` JSON 파싱·`is_expired` 파생 계산 그대로 상속. ✔
- **시세 합성**: `get_quote`를 페이지당 1회 호출(symbol 집합), N+1 없음. asset 미존재 시 `asset=None`, quote 미존재 시 `"0"` — watchlist 패턴 일치. ✔
- **직렬화**: `price`/`change_percent` 문자열 Decimal(테스트 `"195.64"`·`isinstance str` 단언). ✔
- **단일 정의**: `AssetBriefResponse`를 watchlist schema에서 import 재사용(중복 정의 없음). ✔
