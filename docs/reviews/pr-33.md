## Suggestions

### 1. `collect_news_job` — `JobRunService` 인스턴스 중복 생성 (Minor)

**위치:** `app/worker/jobs/news.py:11,14,17`

`JobRunService(db)` 인스턴스가 `start()`, `succeed()`, `fail()` 호출마다 새로 생성된다. 동일 DB 세션을 재사용하므로 기능상 문제는 없으나, 단일 인스턴스로 정리하면 가독성이 개선된다.

```python
svc = JobRunService(db)
job_run = svc.start("news_collection", {"symbols": symbols})
...
svc.succeed(job_run.id)
```

**권장 수정 방향:** 함수 초입에 `svc = JobRunService(db)` 로 단일 인스턴스 생성 후 재사용. `analysis.py`도 동일 패턴.
