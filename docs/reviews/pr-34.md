## Suggestions

### 1. `complete_json` 오류 경로 테스트 누락 (Minor)

**위치:** `tests/test_llm_adapter.py`

`OpenAIClient.complete_json`의 타임아웃·호출 실패·유효하지 않은 JSON·JSON 객체가 아닌 경우가 테스트되지 않았다.

**권장 수정 방향:** `complete` 메서드와 동일한 패턴으로 오류 경로 테스트 추가.

### 2. `MockLLMClient` non-dict 분기 미커버 (Minor)

**위치:** `app/adapters/llm/mock.py:26`

`responses` 값이 `dict`가 아닌 경우의 분기가 테스트되지 않았다. 제거하거나 테스트를 추가하는 것이 명확하다.

**권장 수정 방향:** 테스트 추가 또는 해당 분기 제거.

### 3. `complete_json` schema message prepend 동작 미문서화 (Minor)

**위치:** `app/adapters/llm/openai.py`

호출 측이 이미 system 메시지를 포함하면 system 메시지가 두 개가 된다. 향후 프롬프트가 복잡해질 때 혼선이 생길 수 있다.

**권장 수정 방향:** 주석으로 이 동작을 명시.
