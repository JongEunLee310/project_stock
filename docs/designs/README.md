# Design Docs Index

설계 스켈레톤 문서를 마일스톤별로 묶은 인덱스다. 파일 번호는 작성 순서를 인코딩하며,
파일을 이동하지 않고 이 인덱스로만 버전 묶음을 제공한다. 새 설계 문서를 추가하면
다음 번호를 부여하고 아래 해당 마일스톤 목록에 한 줄을 추가한다.

번호 경계: v0.1은 `005`–`025`, v0.2는 `026`–`045`. `028`은 design 문서가 아니라
task 핸드오프(`.codex/tasks/task-028-frontend-api-spec.md`)로만 존재해 비어 있다.
