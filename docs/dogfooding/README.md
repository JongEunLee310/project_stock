# Dogfooding Results

## Directory Purpose

This directory records the results of dogfooding exercises — end-to-end workflow tests performed on this template before use in real projects.

## Included Documents

- `001-rollback-plan-template.md` — Dogfooding 001: Add Rollback Plan section to Codex Task Packet Template

## File Naming Rules

Use `NNN-short-description.md` format. `NNN` is zero-padded and matches the dogfooding plan task number.

## Related Directories

- `docs/feedback/` — dogfooding plan, handoff drafts, retrospectives
- `docs/failures/` — failure records from dogfooding
- `docs/decisions/` — ADRs triggered by dogfooding findings
