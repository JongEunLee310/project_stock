# Archive

## Directory Purpose

This directory stores deprecated or historical documents that should not guide current work.

## Included Documents

- `deprecated-docs.md`

## Writing Rules

Archived documents must clearly state that they are no longer current.

## File Naming Rules

Use descriptive lowercase kebab-case names.

## Update Criteria

Move documents here when they are historically useful but no longer current.
